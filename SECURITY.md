# Security Policy

## Supported Versions

- `0.1.x`: supported

## Reporting a Vulnerability

1. Do not open a public issue for unpatched vulnerabilities.
2. Send a private report to your internal security channel or responsible maintainer.
3. Include reproduction steps, impacted component, and expected impact.
4. Expect an initial triage response within 3 business days.

## Scope

TCPENT is a defensive assessment framework. Reports about offensive feature requests are out of scope.
